À propos de la visionneuse PowerPoint

La visionneuse Microsoft Office PowerPoint permet de lire des présentations créées dans Microsoft Office PowerPoint 2003, en respectant toutes leurs fonctionnalités. Elle permet l'affichage des présentations créées dans PowerPoint 97, PowerPoint 2000 et PowerPoint 2002. En outre, la visionneuse PowerPoint prend en charge l'ouverture des présentations PowerPoint protégées par mot de passe.

Configuration requise

  • PC équipé d'un processeur de 200 MHz ou plus
  • Microsoft Internet Explorer 5 ou version ultérieure
  • Systèmes d'exploitation pris en charge : Microsoft Windows 98 Deuxième Édition, Windows Millenium Edition, Windows 2000 avec le Service Pack 3, Windows XP ou Windows Server 2003.
  • Mémoire RAM :
    • Pour Windows 98 SE, Windows ME : 32 Mo de RAM
    • Pour Windows 2000 avec le Service Pack 3 : 64 Mo de RAM
    • Pour Windows XP, Windows Server 2003 : 128 Mo de RAM
  • 5 Mo d'espace disque disponible
  • Écran Super VGA (800 x 600) ou de résolution supérieure

Problèmes multilingues

La visionneuse PowerPoint ne prend pas en charge les Outils de vérification linguistique Microsoft Office.

La visionneuse PowerPoint ne prend pas en charge les présentations dans les langues bi-directionnelles, qui ont été créées dans PowerPoint 97.

Retour au Sommaire

Utilisation de la visionneuse PowerPoint

La visionneuse PowerPoint n'affiche pas correctement les animations de diagrammes. Au lieu d'être animé, le diagramme apparaîtra comme terminé.

La visionneuse PowerPoint ne prend pas en charge l'ouverture des présentations IRM (Information Rights Management).

La visionneuse PowerPoint ne prend pas en charge l'exécution de macros, de programmes ou l'ouverture d'objets liés ou intégrés.

La visionneuse PowerPoint ne prend pas en charge l'affichage des formats suivants de graphiques liés à une présentation : .eps, .pct, .emz, .cdr, .cgm, .wpg et .wmz.
